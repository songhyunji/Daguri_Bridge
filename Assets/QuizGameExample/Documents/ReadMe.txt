=======================================
Quiz Game Easy Example & Baseball Scene
=======================================

Thank you for buying this package!
For those who want to make an action quiz game for the first time will help.

Overview:
	- To make it easier for beginners to understand 
	  the short length of the source code was created.
	- Youtube Link : http://youtu.be/Oe7lLneE8f4
	- Demo Game Link : http://qatop.com/webgl/BaseballQuiz/

Testing in Unity Editor: 
	- Run ".../Scenes/Game.unity"!
	- Click answer bar for action!
	- Main InGame code is ".../Scripts/GameManager.cs".
	- The quiz data is ".../Database/quiz_demo_data_asset.txt".
	- Data is separated by Tab code and 5 columns 
	- (quiz number, question, answer x 4, correct number)
	- Two baseball 3d character fbx models are included.
	- Two baseball 2d character are included.
	- Popular baseball stadium fbx model is included.
	- Three background music is included.
	- DOTween plugin motion effect is included.
	- Unity New UI is used.
	- Designed Graphic UI is included.
	- Mecanim humanoid animation is used. 

Build & Run :
    - Configure "Build Setting" sub menu in File Menu.
	- Set Build Platform & Press "Run & Build" Button.
	- Set Your Build Folder.

Note:
	- This app is example for beginners making action quiz game.
	- DOTween free version was used.
	- Two 3D character models with mecanim included.
	- Popular baseball stadium fbx model is included.
